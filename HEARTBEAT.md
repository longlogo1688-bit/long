# HEARTBEAT.md - 小龙的记忆维护任务

## 任务说明
野原需要聊天能连接上下文的技能。通过定期心跳检查，自动整理记忆，确保连续性。

## 执行频率
每 4 小时检查一次
- 额外：每天 23:00 自动保存记忆并执行 Git 上传

## 检查清单

### 1. 记忆文件状态
- [ ] 检查 `MEMORY.md` 是否存在且最新
- [ ] 检查 `memory/YYYY-MM-DD.md` 今日记录是否正常
- [ ] 检查 `IDENTITY.md` 和 `USER.md` 是否完整
- [ ] 保存今日对话，执行 Git 提交并推送到 GitHub

### 2. 技能状态
- [ ] `design-master` 技能是否正常
- [ ] `content-master` 技能是否正常
- [ ] 案例库是否有新内容需要整理

### 3. 对话连续性维护
- [ ] 总结今日重要对话要点
- [ ] 更新野原的偏好变化
- [ ] 记录新的学习成果
- [ ] 归档已完成的事项

### 4. 每日固定任务
- [ ] **每天 12:00**：推送 1-2 条 AI 资讯 + 推荐一本有用的书
- [ ] 长时间未对话时，发送问候
- [ ] 有待办事项时，提醒野原
- [ ] 新技能可用时，介绍功能
- [ ] **每天23:00**：自动保存所有记忆文件，执行 Git 提交并推送到 GitHub

## 输出要求
每次检查后：
1. 更新相关记忆文件
2. 记录检查日志到 `memory/heartbeat-YYYY-MM-DD.log`
3. 如有重要事项，主动告知野原
